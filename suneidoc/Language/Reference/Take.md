### Take

``` suneido
(iterable, n) => sequence
```

Returns a [sequence](<../Basic Data Types/Sequence.md>) containing the first n values from the iterable. Take is normally accessed as object/sequence.Take

For example:

``` suneido
Primes().Take(5)
    => #(2, 3, 5, 7, 11)
```
