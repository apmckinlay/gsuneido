#### Suneido.Info

``` suneido
(string = false) => number
```

Returns a Suneido metric or a list of the available metrics.

These are also available from the web monitor.

See also: [ResourceCounts](<../ResourceCounts.md>), [Suneido.GoMetric](<Suneido.GoMetric.md>)
