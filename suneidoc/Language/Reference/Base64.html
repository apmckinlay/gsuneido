Base64

.Encode(string) => base64_string
.Decode(base64_string) => string

Returns the encoded or decoded string.

For example:

Base64.Encode("hello") => "aGVsbG8="

Base64.Decode("aGVsbG8=") => "hello"

Note: Also available as string.Base64Encode and string.Base64Decode

See also: RFC2045 MIME Part One: Format of Internet Message Bodies
