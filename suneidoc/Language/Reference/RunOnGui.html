Windows Builtin

RunOnGui

(block) => killer

Queues the block to run from the main GUI thread message loop. The block is run from a shared 10ms (the minimum) Windows timer. Timers have low priority so if the UI is busy there may be additional delays.

Note: RunOnGui can only be used from non GUI threads.

killer.Kill() will remove the block from the pending queue if it is still there.

See also: Defer, Delay
