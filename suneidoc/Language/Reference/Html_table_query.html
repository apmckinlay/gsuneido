Html_table_query

(query) => string

Generate an HTML table from a query.

On the query, use project to select and order the columns, and sort to order the rows.

For example:

Html_table_query('tables where table < 10')

will return an HTML string that produces:

table#tablenamenextfieldnrowstotalsize
2tables5512864
4columns31593381
6indexes9764672
8triggers30100

The column headings will use the heading from the data dictionary Field_ definitions when available.

See also: Html_table, Xml
