OptContribution

(name, def) => value

Gets Contributions(). Throws an exception if there are more than one. Returns the single contribution, or the specified def value if there are no contributions.

If the contributions are functions, you probably want to make the default a function, for example:

OptContribution(name, function (@args) { <def> })(...)

See also: Contributions, GetContributions, LastContribution, SoleContribution, Plugins
