date.MinusHours

(date) => number

Returns the number of minutes between two dates.

Simply does date.MinusSeconds / 3600 so it may return fractional amounts.

For example:

#20000303.1030.MinusHours(#20000303.0915) => 1.25

Warning: beware of doing this across daylight savings changes.

See also: date.MinusDays, date.MinusMinutes, date.MinusMonths, date.MinusSeconds
