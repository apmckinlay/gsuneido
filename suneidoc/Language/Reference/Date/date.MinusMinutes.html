date.MinusMinutes

(date) => number

Returns the number of minutes between two dates.

Simply does date.MinusSeconds / 60 so it may return fractional amounts.

For example:

#20000303.1030.MinusMinutes(#20000303.1015) => 15

Warning: beware of doing this across daylight savings changes.

See also: date.MinusDays, date.MinusHours, date.MinusMonths, date.MinusSeconds
