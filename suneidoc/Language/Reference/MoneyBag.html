MoneyBag

MoneyBag is used to easily accumulate totals in multiple currencies.

For example:

mb = new MoneyBag
mb.Plus(100, 'CDN')
mb.Minus(100, 'CDN')
mb.Plus(400, 'USD')
mb.Plus(56.78, 'USD')
mb.Amounts()
	=> #(CDN: 0, USD: 456.78)
