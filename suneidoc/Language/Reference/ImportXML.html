ImportXML

(from_file, to_query, fields = false, header = true)

Derived from Import

Class for importing information from an eXtensible Markup Language textfile.

For example:

ExportXML('tables', 'tables.xml')
Database( "CREATE importTablesTest ( table , tablename , nextfield , nrows , totalsize ) 
    KEY( table )" )
ImportXML('tables.xml', 'importTablesTest')

Will take the information from the tables.xml file and output it to the importTablesTest, which will look similar to this:

table tablename nextfield nrows totalsize
2 "tables" 5 45 4726
4 "columns" 3 145 3020
6 "indexes" 9 65 2842
8 "triggers" 3 0 100

Note: This function can only handle XML files that are in the same format as those produced by ExportXML. Any DTD is ignored.

See also: ExportXML
