SmtpClient

(smtp_server, user = false, password = false, 
    timeout = 60, helo_arg = "")

Starts an SMTP session with the given server.

The timeout argument is in seconds. This is the amount of time the SocketClient will wait for a response from the server before timing out. If the timeout is exceeded, an exception will be thrown. This timeout does not apply to the initial establishment of the socket connection.

For example:

sc = SmtpClient("mail.com");
sc.SendMail("me@mail.com", "me@mail.com", "mckinlay@suneido.com",
    "re. hello",
    "this is my message");
sc.Close();

If the server requires authentication, supply a user name and password.

Note: The current version of SmtpClient only handles AUTH LOGIN authentification.

Some servers also require a specific argument to the HELO/EHLO command.

See also: SmtpSendMessage, PopClient, RFC821 Simple Mail Transfer Protocol, RFC2554 SMTP Service Extension for Authentication
