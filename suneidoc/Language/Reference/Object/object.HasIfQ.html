object.HasIf?

(block) => true or false

Returns true if the object contains a value for which the block returns true, false if not.

For example:

#(123, "abc", name: "Fred", age: 23).HasIf?({ String?(it) and it.Capitalized?() }) 
    => true

Note: object.HasIf? is identical to object.Any?

See also: object.Any?, object.Every?, object.Find, object.FindAll, object.FindAllIf, object.FindIf, object.FindLastIf, object.FindOne, object.Has?
