object.Extract

(name [, default]) => value

Combines object.GetDefault and object.Delete

  • if the member exists, its value is returned and the member is deleted
  • if the member doesn't exist
    • if you supplied a default, it is returned
    • otherwise, an exception is thrown

See also: NameArgs, object.ListToMembers, object.ListToNamed, object.Project, object.ProjectValues, object.Slice
