Builtin

object.FindAllIf

(block) => list

Returns a list of all the members that have values for which the block returns true.

For example:

Object(1, 2, a: 1, b: 2).FindAll(1)
    => #(0, a)

See also: object.Any?, object.Every?, object.Find, object.FindAll, object.FindIf, object.FindLastIf, object.FindOne, object.Has?, object.HasIf?
