object.Intersects?

(object) => true or false

Returns true if the two objects have any common values. For example:

#(1, 2, 3).Intersects?(#(4, 5, 6))
    => false

#(1, 2, 3).Intersects?(#(2, 4))
    => true

Intersects? is the opposite of object.Disjoint?

See also: object.Difference, object.Disjoint?, object.Intersect, object.Subset?, object.Union
