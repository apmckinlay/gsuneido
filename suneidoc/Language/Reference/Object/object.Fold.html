object.Fold

(init, block)
(init) { ... }

Calls the block for each member of the object, passing it the accumulated value (starting with init). The block must return the updated accumulated value. This is a left fold. object.Reduce is similar but it takes the first element as the initial value.

For example, to sum the values in an object: (but in practice, use object.Sum)

#(123, 456, 789).Fold(0, {|sum x| sum += x })
    => 1368

Or to concatenate the values in an object: (but in practice, use object.Join)

#("abc", "def", "ghi").Fold("", {|s x| s $= x })
    => "abcdefghi"

Or to average the values: 

ob = #(10, 20, 60).Fold(Object(sum: 0, n: 0))
    { |ob x|
    ob.sum += x
    ++ob.n
    ob
    }
ob.sum / ob.n
    => 30
