object.Disjoint?

(object) => true or false

Returns true if the two objects do not have any common values. For example:

#(1, 2, 3).Disjoint?(#(4, 5, 6))
    => true

#(1, 2, 3).Disjoint?(#(2, 4))
    => false

Disjoint? is the opposite of object.Intersects?

See also: object.Difference, object.Intersect, object.Intersects?, object.Subset?, object.Union
