object.Reduce

(block)

Apply a function of two arguments cumulatively to the values in a list, from left to right, reducing the sequence to a single value. Similar to object.Fold with the first element as the initial value.

Note: The list must have at least one value. If a list has a single value, that value will be the result.

For example: (although in practice you would use object.Sum)

#(1,2,3,4,5).Reduce({|x,y| x + y }) 
    => 15

i.e. ((((1 + 2) + 3) + 4) + 5)

See also: Compose, Curry, Memoize, MemoizeSingle, object.Any?, object.Drop, object.Every?, object.Filter, object.FlatMap, object.Fold, object.Map, object.Map!, object.Map2, object.Nth, object.PrevSeq, object.Take, object.Zip
