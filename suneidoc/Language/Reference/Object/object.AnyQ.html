object.Any?

(callable) => true / false

Returns true if callable returns true for any value in the object, false otherwise.

callable must return true or false. It can be anything that can be called e.g. block, function, class, or instance.

For example:

#(a, b, C, d).Any?({ it.Upper?() })
    => true
#(a, b, c, d).Any?({ it.Upper?() })
    => false

Note: object.Any? is identical to object.HasIf?
