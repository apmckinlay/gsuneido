object.Without

(@values) => object

Returns a new object without any occurrences of the values. It does not modify the original object.

For example:

Object(12, 34, a: 34, b: 78).Without(34)
    => #(12, b: 78)

See also: object.Delete, object.DeleteIf, object.Erase, object.Remove, object.RemoveIf, object.Trim!, object.WithoutFields
