Builtin

object.Unique!

() => this

Deletes adjacent duplicates from the list members of the object (consecutive members starting at 0).

Note: To eliminate all duplicates use object.Sort! first.

For example:

Object(1, 2, 2, 3, 2).Unique!()
    => #(1, 2, 3, 2)

Object(2, 2, 2).Unique!()
    => #(2)

See also: object.UniqueValues
