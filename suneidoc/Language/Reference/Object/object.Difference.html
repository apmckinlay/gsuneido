object.Difference

(object) => object

Returns a new list of the values in this object that are not contained in the passed in object.

Preserves the order of the values.

For example:

#(23, 76, 34, 98, 56).Difference(#(34, 56, 78))
    => #(23, 76, 98)

Note: Member names are ignored and will not be carried over to the result.

See also: object.Disjoint?, object.Intersect, object.Intersects?, object.Subset?, object.Union
