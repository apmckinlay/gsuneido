Builtin

object.FindAll

(value) => list

Returns a list of all the members containing the given value.

For example:

Object(1, 2, a: 1, b: 2).FindAll(1)
    => #(0, a)

See also: object.Any?, object.Every?, object.Find, object.FindAllIf, object.FindIf, object.FindLastIf, object.FindOne, object.Has?, object.HasIf?
