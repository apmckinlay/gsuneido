object.FindLastIf

(callable) => member or false

Searches for the last unnamed list value for which callable returns true. Returns false if the value is not found.

callable is called with a value as its single argument.

callable can be anything that can be called, e.g. block, function, class, or instance.

For example:

#(38, 48, 32, 67, 89).FindLastIf({|x| x < 50}) => 2

Note: Named members are not searched.

See also: object.Any?, object.Every?, object.Find, object.FindAll, object.FindAllIf, object.FindIf, object.FindOne, object.Has?, object.HasIf?
