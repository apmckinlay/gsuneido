object.Map2

(block) => object

See Map2

Note: If there are named members, object.Map2 will include and sort them.

For example:

#(99, c: 12, a: 34, b: 56).Map2({|m,v| m $ ' is ' $ v })
    => #("0 is 99", "a is 34", "b is 56", "c is 12")
