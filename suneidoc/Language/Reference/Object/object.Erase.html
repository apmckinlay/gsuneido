Builtin

object.Erase

(member ...) => object

Erases a member of an object.

Object(12, 34, a: 56, b: 78).Erase('a') => #(12, 34, b: 78)

Unlike Delete, if the member is within the un-named members, the following members are not moved down.

Object(12, 34, 56, 78).Erase(1) => #(12, 2: 56, 3: 78)

Note: Erase modifies the object it is applied to, it does not create a new object.

See also: object.Delete, object.DeleteIf, object.Remove, object.RemoveIf, object.Trim!, object.Without, object.WithoutFields
