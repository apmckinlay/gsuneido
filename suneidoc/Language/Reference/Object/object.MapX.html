object.Map!

(callable) => this

For each member, Map! does:

[member] = callable([member])

callable can be anything that can be called e.g. block, function, class, or instance.

Note: Map! modifies the object it is applied to, it does not create a new object. It cannot be applied to a read-only object. Whereas object.Map creates a new object and can be applied to a read-only object.

For example:

Object(1, 2, 3).Map!({|x| x * 2 }) => #(2, 4, 6)
