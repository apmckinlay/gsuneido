object.FlatMap

(block) => object

Applies the block to each value in this. The callable must return a list of results. These results are combined into a single list similar to object.Flatten.

For example:

twice = function (x) { return [x, x] }
twice(1)
    => [1, 1]
[1, 2].FlatMap(twice)
    => [1, 1, 2, 2]
