object.ListToNamed

(@names) => object

Converts a list of values to an object containing named values. If there are more list values than names they are ignored.

For example:

[1, 2, 3, 4].ListToNamed('a', 'b', 'c')
    => [a: 1, b: 2, c: 3]

See also: NameArgs, object.Extract, object.ListToMembers, object.Project, object.ProjectValues, object.Slice
