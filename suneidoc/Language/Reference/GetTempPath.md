### GetTempPath

``` suneido
() => string
```

Returns the path for the system temporary directory.

For example:

``` suneido
GetTempPath()
    => "C:\Users\Andrew\AppData\Local\Temp\"
```

See also: [GetTempFileName](<GetTempFileName.md>)
