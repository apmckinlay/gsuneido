By

(@fields)

Returns a block that can be used as the comparison function for object.Sort! to sort a list of objects or records by one or more fields. For example:

list = [[name: 'Fred', age: 23], [name: 'Andy', age: 45]]
list.Sort!(By(#age, #name))

=> [[name: "Fred", age: 23], [name: "Andy", age: 45]]

See also: object.Sort!, object.Sorted?, object.SortWith!
