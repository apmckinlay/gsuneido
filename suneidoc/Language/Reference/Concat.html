Concat

(@iterables) => sequence

Returns a sequence containing the elements from the iterables concatenated together.

Concat can also be accessed as object/sequence.Concat

For example:

p = Primes().Take(4)
f = Fibonaccis().Take(4)
Concat(p, f) // or p.Concat(f)
	=> #(2, 3, 5, 7, 0, 1, 1, 2)
