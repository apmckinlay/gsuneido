Type

Type(value) => string

Returns the type of the value. For example:

Type(true) => "Boolean"
Type(123) => "Number"

Types include:

  • Boolean
  • Number
  • String
  • Date
  • Object (includes class instances)
  • Record
  • Function
  • Block
  • Class
  • Method (result of retrieving method from class)
  • Transaction
  • Query
  • Cursor
  • COMobject
  • Builtin
  • BuiltinClass
