number.Floor

() => number

Returns the largest integer not greater than this number.

For example:

n = 123.456;
n.Floor();
    => 123
n = -123.456;
n.Floor();
    => -124

See also: number.Round, number.RoundDown, number.RoundUp, number.RoundToNearest, number.RoundToPrecision, number.Ceiling, number.Int
