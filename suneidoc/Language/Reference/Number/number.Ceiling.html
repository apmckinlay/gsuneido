number.Ceiling

() => number

Returns the smallest integer not less than this number.

For example:

n = 123.456;
n.Ceiling();
    => 124
n = -123.456;
n.Ceiling();
    => -123

See also: number.Round, number.RoundDown, number.RoundUp, number.RoundToNearest, number.RoundToPrecision, number.Floor, number.Int
