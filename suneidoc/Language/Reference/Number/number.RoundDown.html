Builtin

number.RoundDown

(number) => number

Returns this number rounded down to the specified number of decimal places.

(-x).RoundDown(d) is equivalent to -(x.RoundDown(d))

For example:

n = 123.98673
n.RoundDown(3) => 123.986
n.RoundDown(1) => 123.9
n.RoundDown(0) => 123

Negative numbers of decimal places will round to that power of 10. For example:

n = 7654
n.RoundDown(-1) => 7650
n.RoundDown(-2) => 7600

See also: number.Round, number.RoundUp, number.RoundToNearest, number.RoundToPrecision, number.Ceiling, number.Floor, number.Int
