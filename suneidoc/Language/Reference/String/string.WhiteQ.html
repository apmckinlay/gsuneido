string.White?

( ) => true or false

Return true if all characters in the string are whitespace (blank, tab, return, or linefeed) and there is at least one character, false otherwise.

Similar to string.Blank? except that White? returns true for "" (empty string).

For example:

"".White?()
    => false
"a b c".White?()
    => false
"   ".White?()
    => true

See also: string.Alpha?, string.AlphaNum?, string.Blank?, string.Capitalized?, string.Has1of?, string.Has?, string.Lower?, string.Number?, string.Numeric?, string.Prefix?, string.Suffix?, string.Upper?
