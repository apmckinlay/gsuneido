Builtin

string.Find1of

(chars, pos = 0) => number

Returns the first position greater than or equal to pos where a character from chars is found, or Size() if not found.

chars is a list of characters.

If the first character is '^' then the set of characters will be negated. i.e. it will match everything except those characters. A literal '^' can be included as long as it is not the first character. e.g. "abc^"

A range of characters can be abbreviated with '-'. For example 'a-zA-Z'. A literal '-' can be included as the first or last character e.g. "-+*/" or "*/+="

Note: negation and ranges were added 2024-10-25

For example:

"this is a test".Find1of("si") => 2

See also: string.Has1of?

See also: string.FindLast, string.FindLast1of, string.FindRx, string.FindRxLast, string.ForEachMatch, string.ForEach1of, string.Has?, string.Match
