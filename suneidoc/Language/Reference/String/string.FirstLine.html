string.FirstLine

(pos) => line

Returns the first line. i.e. up to the first '\n' with any trailing '\r' removed.

For example:

"one\r\ntwo\nthree".FirstLine()
    => "two"

See also: string.ChangeEol, string.Lines, string.LineAtPosition, string.LineCount, string.LineFromPosition, string.NthLine, string.RemoveBlankLines, string.WrapLines
