string.FindRx

(regexp) => number

Returns the first position within the string where the regexp is matched, or Size() if it's not found.

For example:

"this is a test".FindRx("t[aeiou]") => 10

See also: string.FindLast, string.Find1of, string.FindLast1of, string.FindRxLast, string.ForEachMatch, string.ForEach1of, string.Has?, string.Match
