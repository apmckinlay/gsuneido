string.UniqueChars

() => string

Returns a string containing the unique characters of this string, preserving the order of the characters.

For example:

"abracadabra".UniqueChars()
    => "abrcd"

Follow with string.Divide to get the characters as a list in an object, for example:

"abracadabra".UniqueChars().Divide()
    => #("a", "b", "r", "c", "d")

See also: object.UniqueValues
