Builtin

string.Lower?

() => true or false

Returns true if all letters are lower case (not capitals), else returns false.

For example:

"".Lower?()
    => false
"123".Lower?()
    => false
"Hello".Lower?()
    => false
"hello world 123".Lower?()
    => true

Use string.Upper? to check for lower case.

See also: string.Alpha?, string.AlphaNum?, string.Blank?, string.Capitalized?, string.Has1of?, string.Has?, string.Number?, string.Numeric?, string.Prefix?, string.Suffix?, string.Upper?, string.White?
