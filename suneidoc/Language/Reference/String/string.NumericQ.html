Builtin

string.Numeric?

( ) => true or false

Return true if all characters in the string are numeric (digits) and there is at least one character, false otherwise.

For example:

"".Numeric?()
    => false
"123".Numeric?()
    => true
"123abc".Numeric?()
    => false

To determine if a string is a number use string.Number?

See also: string.Alpha?, string.AlphaNum?, string.Blank?, string.Capitalized?, string.Has1of?, string.Has?, string.Lower?, string.Number?, string.Prefix?, string.Suffix?, string.Upper?, string.White?
