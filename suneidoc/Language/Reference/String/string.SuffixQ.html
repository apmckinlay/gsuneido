Builtin

string.Suffix?

(string) => true or false

Returns true if the string ends with the supplied string, false otherwise.

For example:

"hello world".Suffix?("he") => false
"hello world".Suffix?("world") => true

Use string.Prefix? for starts with

See also: string.Alpha?, string.AlphaNum?, string.Blank?, string.Capitalized?, string.Has1of?, string.Has?, string.Lower?, string.Number?, string.Numeric?, string.Prefix?, string.Upper?, string.White?, string.RemoveSuffix
