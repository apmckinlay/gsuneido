string.BeforeLast

(delimiter) => string

Returns the portion of the string preceding the last occurrence of delimiter.

For example:

str = 'first and second lines but not third and forth lines'
str.BeforeLast('and')
    => "first and second lines but not third "

See also: string.AfterFirst, string.AfterLast, string.BeforeFirst, string.LeftTrim, string.RightTrim, string.Substr, string.Trim
