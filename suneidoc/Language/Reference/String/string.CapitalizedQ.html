string.Capitalized?

() => true or false

Returns true if the first character is upper case, else returns false.

For example:

'aBc'.Capitalized?()
    => false

'Abc'.Capitalized?()
    => true

See also: string.Alpha?, string.AlphaNum?, string.Blank?, string.Has1of?, string.Has?, string.Lower?, string.Number?, string.Numeric?, string.Prefix?, string.Suffix?, string.Upper?, string.White?
