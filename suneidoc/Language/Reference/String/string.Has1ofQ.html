string.Has1of?

(set) => true or false

Returns true if the string contains one or more of the characters in the set string.

For example:

"this is a test".Has1of("qrs") => true

Uses string.Find1of

See also: string.Alpha?, string.AlphaNum?, string.Blank?, string.Capitalized?, string.Has?, string.Lower?, string.Number?, string.Numeric?, string.Prefix?, string.Suffix?, string.Upper?, string.White?
