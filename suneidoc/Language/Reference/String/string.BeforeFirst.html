string.BeforeFirst

(delimiter) => string

Returns a string containing the portion that comes before the first occurrence of delimiter.

For example:

str = 'first and second lines but not third and forth lines'
str.BeforeFirst('lines')
    => "first and second "

See also: string.AfterFirst, string.AfterLast, string.BeforeLast, string.LeftTrim, string.RightTrim, string.Substr, string.Trim
