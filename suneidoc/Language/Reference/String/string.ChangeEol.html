string.ChangeEol

(eol) => line

Replaces line endings ('\n' or '\r\n') with the specified string.

For example:

"one\ntwo\r\nthree".ChangeEol('\r\n').Escape()
    => "one\r\ntwo\r\nthree"

See also: string.FirstLine, string.Lines, string.LineAtPosition, string.LineCount, string.LineFromPosition, string.NthLine, string.RemoveBlankLines, string.WrapLines
