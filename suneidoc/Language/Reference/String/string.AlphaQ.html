Builtin

string.Alpha?

( ) => true or false

Return true if all characters in the string are alphabetic (letters) and there is at least one character, false otherwise.

For example:

"".Alpha?()
    => false
"abc".Alpha?()
    => true
"abc123".Alpha?()
    => false

See also: string.AlphaNum?, string.Blank?, string.Capitalized?, string.Has1of?, string.Has?, string.Lower?, string.Number?, string.Numeric?, string.Prefix?, string.Suffix?, string.Upper?, string.White?
