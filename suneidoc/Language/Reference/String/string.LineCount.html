string.LineCount

() => number

Returns the number of lines in the string.

For example:

"".LineCount()
    => 0

"hello world".LineCount()
    => 1

"hello world\n".LineCount()
    => 1

"hello\nworld".LineCount()
    => 2

"hello\nworld\n".LineCount()
    => 2

See also: string.FirstLine, string.ChangeEol, string.Lines, string.LineAtPosition, string.LineFromPosition, string.NthLine, string.RemoveBlankLines, string.WrapLines
