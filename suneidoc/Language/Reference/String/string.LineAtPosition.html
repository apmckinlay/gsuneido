string.LineAtPosition

(pos) => line

Returns the line containing the specified position.

For example:

"one\ntwo\r\nthree".LineAtPosition(5)
    => "two"

See also: string.FirstLine, string.ChangeEol, string.Lines, string.LineCount, string.LineFromPosition, string.NthLine, string.RemoveBlankLines, string.WrapLines
