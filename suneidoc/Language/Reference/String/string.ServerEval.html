Deprecated Builtin

string.ServerEval

() => value

Evaluates the string as Suneido code on the server.

For example:

"123 + 456".ServerEval() => 579

Warning: Any newlines in the string are silently replaced by blanks.

If you're running standalone, string.ServerEval is identical to string.Eval.

Note: string.ServerEval is deprecated and may be removed in the future. Use ServerEval instead
