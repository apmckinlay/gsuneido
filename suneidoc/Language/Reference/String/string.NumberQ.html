Builtin

string.Number?

() => true or false

Returns true if the string is a valid number, false otherwise.

For example:

"-123.456".Number?() => true
"hello world".Number?() => false

See also: string.Alpha?, string.AlphaNum?, string.Blank?, string.Capitalized?, string.Has1of?, string.Has?, string.Lower?, string.Numeric?, string.Prefix?, string.Suffix?, string.Upper?, string.White?
