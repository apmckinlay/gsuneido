string.Lines

( ) => object

Returns a Sequence of newline separated lines from the string. Return characters are not skipped. A newline on the last line is optional. An empty string gives no lines.

For example:

s = "first line\nsecond line\r\nthird line\n"
s.Lines()
    => #("first line", "second line", "third line")

Note: To process the lines of a file, consider FileLines or file.Readline rather than reading the whole file into memory with GetFile(...).Lines()

See also: string.FirstLine, string.ChangeEol, string.LineAtPosition, string.LineCount, string.LineFromPosition, string.NthLine, string.RemoveBlankLines, string.WrapLines
