Builtin

string.Find

(string, pos = 0) => number

Returns the first position greater than or equal to pos where string is found, or Size() if not found.

For example:

"this is a test".Find("is") => 2
"hello world".Find("o") => 4
"hello world".Find("o", 5) => 7
"hello world".Find("x") => 11

See also: string.FindLast, string.Find1of, string.FindLast1of, string.FindRx, string.FindRxLast, string.ForEachMatch, string.ForEach1of, string.Has?, string.Match
