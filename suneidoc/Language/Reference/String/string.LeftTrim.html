string.LeftTrim

(chars = " \t\r\n") => string

Returns a copy of the string with leading characters removed. The characters to be removed default to space, tab, return, and linefeed.

For example:

"   hello world   ".LeftTrim()
    => "hello world   "

See also: string.AfterFirst, string.AfterLast, string.BeforeFirst, string.BeforeLast, string.RightTrim, string.Substr, string.Trim
