string.Trim

(chars = " \t\r\n") => string

Returns a copy of the string with leading and trailing characters removed. The characters to be removed default to space, tab, return, and linefeed.

For example:

"   hello world   ".Trim()
    => "hello world"

See also: string.AfterFirst, string.AfterLast, string.BeforeFirst, string.BeforeLast, string.LeftTrim, string.RightTrim, string.Substr
