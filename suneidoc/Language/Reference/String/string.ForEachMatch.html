string.ForEachMatch

(regex, block)

Calls the block for each non-overlapping match in the string. Passes the result of string.Match to the block

For example:

"ab cab abc def".ForEachMatch("ab") {|m| Print(m) }

=>  #(#(0, 2))
    #(#(4, 2))
    #(#(7, 2))

See also: string.FindLast, string.Find1of, string.FindLast1of, string.FindRx, string.FindRxLast, string.ForEach1of, string.Has?, string.Match
