string.ForEach1of

(chars, block) => number

Calls block with each position that is one of chars.

For example:

"hello world".ForEach1of("abcde")
	{ Print(it) }
=>	1
	10

See also: string.FindLast, string.Find1of, string.FindLast1of, string.FindRx, string.FindRxLast, string.ForEachMatch, string.Has?, string.Match
