string.RemoveBlankLines

() => string

Returns the string with blank lines (zero or more spaces or tabs) removed.

For example:

"one\n \r\n \nthree".RemoveBlankLines().Escape()
    => "one\nthree"

See also: string.FirstLine, string.ChangeEol, string.Lines, string.LineAtPosition, string.LineCount, string.LineFromPosition, string.NthLine, string.WrapLines
