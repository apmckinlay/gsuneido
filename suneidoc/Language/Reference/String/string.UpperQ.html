Builtin

string.Upper?

( ) => true or false

Returns true if all letters in the string are upper case (capitals), else returns false.

For example:

"".Upper?()
    => false
"123".Upper?()
    => false
"Hello".Upper?()
    => false
"HELLO WORLD 123".Upper?()
    => true

Use string.Lower? to check for lower case.

See also: string.Alpha?, string.AlphaNum?, string.Blank?, string.Capitalized?, string.Has1of?, string.Has?, string.Lower?, string.Number?, string.Numeric?, string.Prefix?, string.Suffix?, string.White?
