string.RightFill

(minSize, char = " ") => string

Returns a string padded on the right with blanks's (or the specified character) to be not less than minSize characters. If the string is already minSize or greater, no characters are added.

For example:

x = "hello"
x.RightFill(8, '*')
    => "hello***"

See also: string.Center, string.LeftFill, number.Pad
