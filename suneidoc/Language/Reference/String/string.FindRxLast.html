string.FindRxLast

(regexp) => number or false

Returns the last position in the string where the regexp is matched, or false if it's not found.

For example:

"this is a test".FindRxLast("[aeiou]s") => 11

See also: string.FindLast, string.Find1of, string.FindLast1of, string.FindRx, string.ForEachMatch, string.ForEach1of, string.Has?, string.Match
