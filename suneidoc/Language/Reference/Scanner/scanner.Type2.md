#### scanner.Type2

``` suneido
() => string
```

Returns the type of the current token as a string.

-	"ERROR"
-	"IDENTIFIER"
-	"NUMBER"
-	"STRING"
-	"WHITESPACE"
-	"COMMENT"
-	"NEWLINE"
-	""


Type2 is intended to eventually replace Type.
