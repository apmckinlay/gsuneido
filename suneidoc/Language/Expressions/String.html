String

Operators:

$concatenate, returns a string
=~matches regular expression, returns true or false
!~does not match regular expression, returns true or false

For example:

"hello" $ " " $ "world"
    => "hello world"

Concatenation ($) will automatically convert booleans, numbers, and instances with a ToString method to strings. Using a string operator with other types will throw an exception of "cannot convert {type} to string". Use Display to convert other values to strings.
