### Assignment

|  |  | 
| :---- | :---- |
| = | assignment | 
| += | addition | 
| -= | subtraction | 
| *= | multiplication | 
| /= | division | 
| %= | modulus (remainder) | 
| &= | bitwise and | 
| |= | bitwise or | 
| ^= | bitwise exclusive or | 
| <<= | left shift | 
| >>= | right shift | 


`x op= y ` is equivalent to `x = x op y`
