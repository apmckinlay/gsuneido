Compiler Error

too many local variables

The normal solution to this is to split up your function, often a good idea for readability anyway. Another alternative is to use an object to store a collection of variables.

Instead of:

a = "abra"
b = "cada"
c = "bra"
...

Use:

ob = Object()
ob.a = "abra"
ob.b = "cada"
ob.c = "bra"
...
