Tools

WorkSpace - run code and access other tools
LibraryView - view and edit source code
QueryView - execute database queries and requests
SchemaView - view database table information
ClassView - view the class hierarchy
Getting Around - ways to navigate your code
BookEdit - edit application books
Books - open (run) an application book
Access - view or edit a table or query a record at a time
Browse - view or edit a table or query as a tabular list
TestRunner - run unit tests
Reporter - run reports

Scheduler - schedule tasks to be run periodically
Version Control - track versions of library source code
Language Translation - run the IDE in other languages (e.g. Spanish, Italian, French, German)

Server Monitor - the web-based monitor for Suneido servers
