InfoControl

(def = 0, mandatory = false)

A composite control consisting of a ComboBox to choose the type of information, and a second field to enter the information. The initial type can be specified with def as a number from 0 to 7.

The allowed types and the corresponding controls are:

Work:Phonework phone number
Fax:Phonefax phone number
Home:Phonehome phone number
Cell:Phonecell phone number
Pager:Phonepager phone number
Email:MailLinkemail address
Website:HttpLinkwebsite address
Other:Fieldanything

The value of the control is a string of the form "type: value", for example: "Work: 249-5050". If the right hand field is empty, the value is "". If an InfoControl is set to "", the def type will be selected.

If the mandatory parameter is true, then must enter a value in the field
