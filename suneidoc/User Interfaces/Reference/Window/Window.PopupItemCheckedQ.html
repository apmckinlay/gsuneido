Window.PopupItemChecked?

(hsubmenu, itempos = 0) => true or false or -1
hsubmenu
is a menu handle
itempos
is the item position in the submenu (0,1,2 ...)

Returns true if checked, false if not. If it fails it return -1

See also: Window.CheckPopupItem, Window.EnablePopupItem, Window.GetPopupItemInfo, Window.PopupItemEnabled?, Window.SetPopupItemInfo, window.GetPopupHandle
