### NameAbbrevControl

``` suneido
( prefix = "", suffix = "" )
```

Displays fields for `(prefix $ '_name' $ suffix)`
and `(prefix $ '_abbrev' $ suffix)`
with "(optional)" after the abbreviation field.

For example:

``` suneido
Window(#(NameAbbrev))
```
