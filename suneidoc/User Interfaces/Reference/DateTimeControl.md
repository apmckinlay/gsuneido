### DateTimeControl

``` suneido
() => date
```

Uses [MonthCalControl](<MonthCalControl>) and 
[ListBoxControl](<ListBoxControl>) to allow the 
user to select a date and a time.

For example:

``` suneido
DateTimeControl()
```
