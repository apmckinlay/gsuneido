InfoFormat

( prefix = '', data = False )

Prints any "info" fields (with the specified prefix) in the data vertically, in a column on a report.

For example:

record = #(info2: 'Work: 222-7856', info3: 'Home: 123-4567', info5: 'Email: joe@mail.com')
Params.On_Preview(Object('Info' data: record))
