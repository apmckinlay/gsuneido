report.GetParamsWhere

(field, ...) => string

Calls GetParamsWhere, passing .Params as the data.

For example:

...
Params:
    #(Form
        (ParamsSelect firstname)
        (ParamsSelect lastname))
QueryFormat
    {
    Query()
        {
        return 'mycontacts' $ _report.GetParamsWhere('firstname, lastname')
        }
...

See also: Params, Going Further - Report Parameters
