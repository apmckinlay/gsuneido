times (product)

query times query

The result of times is a table with all the columns from both queries and with the set of rows formed by combining each possible pair of rows from the queries.

Note: The queries cannot have any columns in common.

a b
one 12
two 34
times
c d
three 56
four 78
=
a b c d
one 12 three 56
one 12 four 78
two 34 three 56
two 34 four 78
