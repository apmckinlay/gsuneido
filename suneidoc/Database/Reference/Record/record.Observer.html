Builtin

record.Observer

( observer )

Registers observer to be called whenever a member is modified. Multiple observers can be registered on the same record. Observers are called in the same order they were registered.

Observers are also triggered by the invalidation of rules.

The observer call is equivalent to:

record.Eval(observer, member: member)

Note: this means that if the observer wants to receive the member its parameter must be called "member".

Note: If the observer is a method of a class, it will be called normally, and "this" will be the instance or class, not the record. (This is because when you get a reference to a method, the value you get is "bound" to the class or instance that you got it from.)

For example:

f = function (member) { Print(member $ " changed to " $ this[member]) }
r = new Record
r.Observer(f)
r.a = 1
r.b = 2

would print:

a changed to 1
b changed to 2

See also: record.RemoveObserver
