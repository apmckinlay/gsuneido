Builtin

record.RemoveObserver

( observer )

Remove an observer from a record that was added with record.Observer

For example:

f = function (member) { Print(member $ " changed to " $ this[member]) }
r = new Record
r.Observer(f)
r.a = 1
r.RemoveObserver(f)
r.b = 2

would print:

a changed to 1

Notice that it does not print "b changed to 2".
