QueryExists?

(query [, field: value ...]) => true or false

Returns true if the query contains any records, otherwise returns false. Executes the query in a standalone read-only transaction.

Named arguments (field: value) add a where to the query. For example:

('tables', tablename: value)

is equivalent to:

('tables where tablename = ' $ Display(value)

Note: After BuiltDate 2025-04-17, QueryExists? will replace QueryEmpty?

For example:

QueryExists?("stdlib where num is 2")
=> true

See also: Query1, QueryEmpty?, QueryFirst, QueryLast, transaction.Query1, transaction.QueryEmpty?, transaction.QueryExists?, transaction.QueryFirst, transaction.QueryLast
